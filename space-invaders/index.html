DISPLAY SCREEN
Portrait digital signage PWA. Shows the game + QR code for customers to scan. Open this in a fullscreen browser window.

MOBILE CONTROLLER
Arcade controller loaded on the customer's phone after scanning the QR code. Sends left/right/fire inputs to the display.
💥
GAME OVER SCREEN
Shown on the customer's phone after the game ends. Displays final score, high score, and store CTAs.

▶ SAME-BROWSER DEMO (NO SERVER NEEDED)

  1. Open the Display Screen — note the Session ID shown below the QR code
  2. Open the Mobile Controller in a new tab in the same browser
  3. Append ?session=XXXXXXXX to the mobile URL (use the Session ID from step 1)
  4. The two screens connect automatically via BroadcastChannel
  5. Use arrow keys or the on-screen buttons to play!
